Datensätze

Die Datensätze werden in Tabellen präsentiert, wie in der Beispieltabelle unten. Jede Zelle kann eine 0 oder eine 1 beinhalten. In der ersten Spalte sehen Sie 8 Einträge von der Hauptbeobachtung (HB); in Spalte 2 bis 4 sehen Sie jeweils 9 Einträge für 3 Nebenbeobachtungen (NB). Die Einträge in den einzelnen Zeilen stellen Beobachtungen dar, die jeweils gleichzeitig gemacht wurden. Beachten Sie, dass kein zeitlicher Zusammenhang zwischen den einzelnen Zeilen besteht. Es ist also nicht so, dass die Ereignisse, die zu den Einträgen in der ersten Zeile geführt haben, notwendigerweise vor den Einträgen in der 9. Zeile stattgefunden haben. In der 9. Zeile fehlt die Hauptbeobachtung, Sie sehen allerdings die Nebenbeobachtungen. Zwischen Haupt- und Nebenbeobachtungen kann ein kausaler Zusammenhang bestehen; dies muss aber nicht so sein. Ob kausal oder nicht, der Zusammenhang zwischen den Beobachtungen ist in jeder Zeile derselbe.

Ihre Aufgabe

Ihre Aufgabe ist es einzuschätzen, ob hinter dem Fragezeichen (?) eine 0 oder eine 1 versteckt ist. Dabei sehen Sie die Datentabelle. Zusätzlich dazu erhalten Sie eine kurze Erklärung von einem Sender, die Sie auf einen möglichen Zusammenhang in den Daten hinweist. Diese Erklärungen können Ihnen dabei helfen einzuschätzen, ob hinter dem Fragezeichen (?) eine 0 oder eine 1 versteckt ist. Die Erklärungen formulieren einen möglichen Zusammenhang zwischen Haupt- und Nebenbeobachtungen. Dabei kann es sich um den wahren Zusammenhang handeln; es muss aber nicht so sein. Zum Beispiel könnte es auch Zufall sein, dass die Hauptbeobachtung (HB) und eine der Nebenbeobachtungen (NB) gleichzeitig gleich 1 sind. Es könnte aber auch sein, dass eine der Beobachtungen gleich 1 ist, weil eine andere Beobachtung auch gleich 1 ist. Zu jedem Datensatz sehen Sie jeweils eine einzelne Erklärung.

Sie erhalten eine sichere Auszahlung von 1,50€ für Ihre Teilnahme. Zusätzlich dazu erhalten Sie eine Auszahlung von bis zu 3€, die von Ihren Antworten im Experiment abhängt. Sie können als zwischen 1,50€ und 4,50€ verdienen.

Hinter dem Fragezeichen (?) ist entweder eine 1 oder eine 0 versteckt. Ihre Auszahlung hängt von der von Ihnen angegebenen Wahrscheinlichkeit ab und davon, ob sich hinter dem Fragezeichen (?) tatsächlich eine 1 verbirgt. Die Auszahlung wird so berechnet, dass es für Sie zur höchsten erwarteten Auszahlung führt, wenn Sie Ihre echte Einschätzung der Wahrscheinlichkeit angeben.

Details zur Auszahlung: Im Folgenden bezeichne q die von Ihnen angegebene Wahrscheinlichkeit, dass sich hinter dem Fragezeichen (?) eine 1 verbirgt. Nachdem Sie q angegeben haben, wird mithilfe des wahren Wertes hinter dem Fragezeichen (?) Ihre Auszahlung durch eine Lotterie entschieden. Sollte in der Tat eine 1 hinter dem Fragezeichen (?) sein, erhalten Sie eine Auszahlung von 3€ mit einer Wahrscheinlichkeit von 1-(1-q)*(1-q). Sollte sich dort stattdessen eine 0 verbergen, so erhalten Sie 3€ mit einer Wahrscheinlichkeit von 1-q*q.

Zusätzlich zu Ihrer Einschätzung der Wahrscheinlichkeit fragen wir Sie nach jeder Entscheidung, wie sicher Sie sich sind, dass Ihre Einschätzung zutreffend ist.
