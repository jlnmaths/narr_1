Hinter dem Fragezeichen (?) ist entweder eine 1 oder eine 0 versteckt. Ihre Auszahlung hängt von der von Ihnen angegebenen Wahrscheinlichkeit ab und davon, ob sich hinter dem Fragezeichen (?) tatsächlich eine 1 verbirgt. Die Auszahlung wird so berechnet, dass es für Sie zur höchsten erwarteten Auszahlung führt, wenn Sie Ihre echte Einschätzung der Wahrscheinlichkeit angeben. Sie erhalten außerdem einen Bonus von 1 Unit, sollte sich hinter dem Fragezeichen (?) eine 1 verbergen.

Details zur Auszahlung: Im Folgenden bezeichne q die von Ihnen angegebene Wahrscheinlichkeit, dass sich hinter dem Fragezeichen (?) eine 1 verbirgt. Nachdem Sie q angegeben haben, wird mithilfe des wahren Wertes hinter dem Fragezeichen (?) Ihre Auszahlung durch eine Lotterie entschieden. Sollte in der Tat eine 1 hinter dem Fragezeichen (?) sein, erhalten Sie eine Auszahlung von 1 Unit mit einer Wahrscheinlichkeit von 1-(1-q)*(1-q). Sollte sich dort stattdessen eine 0 verbergen, so erhalten Sie 1 Unit mit einer Wahrscheinlichkeit von 1-q*q.

Zusätzlich zu Ihrer Einschätzung der Wahrscheinlichkeit fragen wir Sie nach jeder Entscheidung, wie sicher Sie sich sind, dass Ihre Einschätzung zutreffend ist.
